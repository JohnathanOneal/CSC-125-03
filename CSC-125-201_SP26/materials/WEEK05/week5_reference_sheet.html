Quick Reference

Basic if Statement

An if statement runs its indented block only when the condition is True. The condition line must end with a colon, and the body must be indented by 4 spaces.

temperature = 95

if temperature > 90:
    print("It is hot outside")

print("Have a good day")  # Always runs (not indented under if)

Any code that is not indented under the if block will always run, regardless of whether the condition was True or False.

if-else Statement

An if-else provides two branches. Exactly one block always runs: the if block when the condition is True, or the else block when it is False.

number = 7

if number % 2 == 0:
    print(f"{number} is even")
else:
    print(f"{number} is odd")
if-elif-else Chain

Use elif (short for "else if") to check multiple conditions in sequence. Python evaluates each condition from top to bottom and runs the first block whose condition is True. All remaining branches are skipped.

score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"Your grade is {grade}")  # Your grade is B

The else block at the end is optional but acts as a catch-all for any value that did not match an earlier condition. Order matters: place the most specific conditions first.

Logical Operators

Logical operators combine or modify boolean expressions.

Operator Meaning Example Result
and Both must be True True and False False
or At least one must be True True or False True
not Reverses True/False not True False

Combining Conditions

Use logical operators inside if statements to test multiple conditions at once. Parentheses improve readability in complex expressions.

age = 25
income = 50000

# and: both conditions must be True
if age >= 18 and income >= 30000:
    print("Eligible for credit card")

# or: at least one condition must be True
day = "Saturday"
if day == "Saturday" or day == "Sunday":
    print("It is the weekend")

# not: reverses the result
logged_in = False
if not logged_in:
    print("Please log in to continue")

# Parentheses for clarity in complex conditions
temperature = 72
humidity = 40
if (temperature >= 65 and temperature <= 80) and (humidity < 60):
    print("The weather is pleasant")
Nested Conditionals

A nested conditional places one if statement inside another. Use this when a second decision depends on the result of the first. Each level adds 4 more spaces of indentation.

has_ticket = True
age = 15

if has_ticket:
    if age >= 18:
        print("Welcome to the R-rated movie")
    else:
        print("You need a parent for this movie")
else:
    print("Please purchase a ticket first")

Tip: keep nesting to 2-3 levels at most. Deeply nested code becomes difficult to read and debug. Consider combining conditions with and as an alternative when possible.

Indentation Rules
  • Use 4 spaces per indentation level (not tabs)
  • A colon (:) is required after every if, elif, and else
  • All lines in the same block must have the same indentation
  • Empty lines are allowed within blocks without affecting indentation
  • Code that should run after a conditional block must return to the previous indentation level
x = 10
if x > 5:
    print("x is large")     # 4 spaces - inside if block
    print("still inside")   # 4 spaces - same block

print("always runs")        # 0 spaces - outside if block
Common Errors
Error Cause Fix
SyntaxError: missing colon Forgot : after condition Add : at end of if/elif/else
IndentationError: unexpected indent Inconsistent spacing Use 4 spaces consistently per level
Logic error: always True/False Using = instead of == Use == for comparison, = for assignment
Logic error: wrong branch runs elif order checks lower threshold first Order conditions from most specific to least 
# Wrong: checks lower threshold first
if score >= 60:
    grade = "D"   # score of 95 would get "D"
elif score >= 90:
    grade = "A"   # This never runs for 95

# Correct: check highest threshold first
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 60:
    grade = "D"
