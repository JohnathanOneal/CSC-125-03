Answers

  1. Q1: b) GoGoGo — string repetition with * repeats the string 3 times with no spaces between copies
  2. Q2: b) 11 — PEMDAS: multiplication first (4 × 2 = 8), then addition (3 + 8 = 11); choice a) 14 is the left-to-right mistake
  3. Q3: b) <class 'str'> — "72" is in quotes, so it is a string regardless of its numeric-looking content
  4. Q4: input — input() displays a prompt and returns whatever the user types as a string
  5. Q5: int — int() converts the string "88" to a whole number so it can be added to 12
  6. Q6: Cross out favourit_color, write favorite_color — NameError: the variable name is misspelled (missing the e in favorite)
  7. Q7: Cross out print("Hello World", write print("Hello World") — SyntaxError: missing closing parenthesis
  8. Q8: Cross out age (in the print), write str(age) — TypeError: cannot concatenate a string and an integer; must convert age to a string first
