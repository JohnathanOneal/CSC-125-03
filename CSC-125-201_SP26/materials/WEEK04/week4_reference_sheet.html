Division Types

Python has three division-related operators. Regular division always returns a float. Floor division rounds down to a whole number. Modulo returns the remainder.

10 / 3    # 3.3333... (always float)
10 // 3   # 3 (rounds down to whole number)
10 % 3    # 1 (remainder)
# Relationship: 10 = (10 // 3) * 3 + (10 % 3)
Practical Uses of // and %

Floor division and modulo are commonly used for time conversions, checking divisibility, and extracting digits.

# Time conversion
total_minutes = 135
hours = total_minutes // 60      # 2
minutes = total_minutes % 60     # 15

# Even/odd check
number = 17
number % 2 == 0    # False (odd)
number % 2 != 0    # True (odd)
Operator Precedence

Python follows standard math order of operations (PEMDAS). When in doubt, use parentheses to make your intent clear.

Priority Operator(s) Description
1 (highest) () Parentheses
2 ** Exponentiation
3 *, /, //, % Multiplication, Division, Floor Div, Modulo
4 +, - Addition, Subtraction
5 ==, !=, <, >, <=, >= Comparisons
6 (lowest) =, +=, -=, etc. Assignment 
2 + 3 * 4          # 14 (not 20)
(2 + 3) * 4        # 20

# Use parentheses for clarity
average = (score1 + score2 + score3) / 3
total = price * (1 + tax_rate)
Comparison Operators

Comparison operators compare two values and return a boolean (True or False).

Operator Meaning Example Result
== Equal to 20 == 18 False
!= Not equal to 20 != 18 True
< Less than 20 < 65 True
> Greater than 20 > 18 True
<= Less than or equal to 20 <= 21 True
>= Greater than or equal to 20 >= 18 True 
age = 20
age == 18     # False
age != 18     # True
age > 18      # True
age < 65      # True
age >= 18     # True
age <= 21     # True

# Chained comparisons (Python feature)
18 <= age <= 30    # True
Boolean Expressions

Comparisons return boolean values that can be stored in variables for later use.

age = 20
score = 85

can_vote = age >= 18           # True
is_passing = score >= 70       # True
is_perfect = score == 100      # False
is_teen = 13 <= age <= 19      # False

print(f"Can vote: {can_vote}")
print(f"Passing: {is_passing}")
Assignment Operators

Assignment operators update a variable by combining an arithmetic operation with assignment.

Operator Example Equivalent To
+= score += 10 score = score + 10
-= score -= 3 score = score - 3
*= score *= 2 score = score * 2
/= score /= 7 score = score / 7 
score = 0
score += 10    # score = score + 10 → 10
score -= 3     # score = score - 3 → 7
score *= 2     # score = score * 2 → 14
score /= 7     # score = score / 7 → 2.0
Common Errors

Here are common errors you may encounter when working with math operations and operators.

Error Cause Fix
Wrong result Missing parentheses in expressions Use () to control operator precedence
TypeError Doing math with strings from input() Convert with int() or float() first
Confusion Using = instead of == for comparison = assigns, == compares
ZeroDivisionError Dividing by zero Check the divisor before dividing
