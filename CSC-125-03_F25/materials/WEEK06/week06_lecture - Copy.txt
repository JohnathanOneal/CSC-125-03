    "## 6. Tuples - The Immutable Cousin\n",
    "\n",
    "Tuples use parentheses `()` and **cannot be changed** after creation."
   ],
   "metadata": {}
  },
  {
   "cell_type": "code",
   "source": [
    "# Creating tuples\n",
    "point = (10, 20)\n",
    "rgb = (255, 128, 0)\n",
    "person = (\"Alice\", 25, \"Engineer\")\n",
    "\n",
    "print(f\"Point: {point}\")\n",
    "print(f\"RGB: {rgb}\")\n",
    "print(f\"Person: {person}\")\n",
    "print()\n",
    "\n",
    "# Indexing works the same\n",
    "name = person[0]\n",
    "age = person[1]\n",
    "print(f\"Name: {name}, Age: {age}\")\n",
    "print()\n",
    "\n",
    "# Tuple UNPACKING - very useful!\n",
    "x, y = point\n",
    "print(f\"x = {x}, y = {y}\")\n",
    "\n",
    "name, age, job = person\n",
    "print(f\"{name} is {age}, works as {job}\")\n",
    "print()\n",
    "\n",
    "# Swapping values\n",
    "a = 5\n",
    "b = 10\n",
    "print(f\"Before: a={a}, b={b}\")\n",
    "a, b = b, a  # Swap!\n",
    "print(f\"After: a={a}, b={b}\")\n",
    "print()\n",
    "\n",
    "# Tuples CANNOT be modified\n",
    "try:\n",
    "    point[0] = 15  # This will error!\n",
    "except TypeError as e:\n",
    "    print(f\"Error: {e}\")\n",
    "    print(\"Tuples are immutable!\")"
   ],

    "## 7. Lists vs Tuples\n",
    "\n",
    "**Lists `[]`:**\n",
    "- Can be changed (mutable)\n",
    "- Use when data needs to change\n",
    "- Examples: shopping cart, game scores, user inputs\n",
    "\n",
    "**Tuples `()`:**\n",
    "- Cannot be changed (immutable)\n",
    "- Use when data is fixed\n",
    "- Examples: coordinates (x, y), RGB colors (255, 0, 0), dates"
   ],
   "metadata": {}

    "## 9. Key Takeaways\n",
    "\n",
    "**Lists:**\n",
    "- Store multiple values: `[1, 2, 3]`\n",
    "- Access with indexing: `my_list[0]` or `my_list[-1]`\n",
    "- Extract with slicing: `my_list[1:3]`\n",
    "- Methods: `.append()`, `.pop()`, `.remove()`, `len()`\n",
    "- Lists are **mutable** (changeable)\n",
    "\n",
    "**Tuples:**\n",
    "- Fixed collections: `(1, 2, 3)`\n",
    "- Same indexing as lists\n",
    "- Cannot be modified (**immutable**)\n",
    "- Use unpacking: `x, y = point`\n",
