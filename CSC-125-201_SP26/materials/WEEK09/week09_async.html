Quick Reference

# While loop pattern
while condition:
    # work
    # update variables so condition eventually becomes False

# Input validation loop
value = input("Enter 1-10: ")
while not value.isdigit() or not (1 <= int(value) <= 10):
    print("Invalid. Try again.")
    value = input("Enter 1-10: ")
print(f"OK: {value}")

# break and continue
n = 0
while n < 10:
    n += 1
    if n == 5:
        continue  # skip this iteration
    if n == 8:
        break     # exit loop early
    print(n)

# try except - specific first
try:
    x = int(input("Number: "))
    print(10 / x)
except ValueError:
    print("Enter an integer.")
except ZeroDivisionError:
    print("Cannot divide by zero.")

# catch-all - useful during debugging
try:
    risky()
except Exception as e:
    print(f"Error: {e}")

Common Pitfalls

Infinite Loops

# Bug - never updates count
count = 0
while count < 3:
    print("Hello")
    # missing: count += 1

Fix: always update loop variables inside the loop or in all code paths.

Converting Input Without Checking

# Bug - may raise ValueError
age = int(input("Age: "))  # user might type "ten"

Fix: validate first or wrap in try except.

Error Types You Should Recognize

  • SyntaxError - invalid Python syntax like missing colon
  • TypeError - operation on wrong type like "a" + 5
  • ValueError - bad value like int("cat")
  • ZeroDivisionError - divide by zero

Mini Examples

1) Input Guard

Ask for a score 0-100 until valid then print pass or fail.

score = input("Score 0-100: ")
while not score.isdigit() or not (0 <= int(score) <= 100):
    print("Try again.")
    score = input("Score 0-100: ")
print("Pass" if int(score) >= 60 else "Fail")

2) Sum Until Stop

Keep adding integers from input. Stop when user types stop. Ignore non-integers gracefully.

total = 0
while True:
    s = input("Enter int or 'stop': ").strip().lower()
    if s == "stop":
        break
    try:
        total += int(s)
    except Exception as e:
        print(f"Ignored: {e}")
print(f"Total: {total}")

3) Skip Multiples

Print numbers 1 to 20 but skip multiples of 4. Stop entirely at 17.

n = 0
while n < 20:
    n += 1
    if n % 4 == 0:
        continue
    if n == 17:
        break
    print(n)
