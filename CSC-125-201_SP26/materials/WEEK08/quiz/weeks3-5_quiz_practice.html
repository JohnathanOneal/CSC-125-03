Answers

  1. Q1: a) 3m 20s - floor division (//) gives the whole number of minutes (200 // 60 = 3), and modulo (%) gives the remaining seconds (200 % 60 = 20).
  2. Q2: a) bob smith - .strip() removes the leading and trailing spaces first, then .lower() converts all letters to lowercase.
  3. Q3: a) Admitted - the or operator only needs one condition to be True. Since age >= 18 is True (20 >= 18), the entire condition is True regardless of has_ticket.
  4. Q4: -= - the assignment operator -= subtracts 15 from the current value of score (100 - 15 = 85).
  5. Q5: f - the f prefix before a string makes it an f-string, which allows variables inside curly braces to be replaced with their values.
  6. Q6: Cross out mesage, write message - NameError: the variable name is misspelled. Python treats mesage as a different, undefined variable.
  7. Q7: Indent print("You passed") by 4 spaces so it is inside the if block - IndentationError: Python requires the code inside an if statement to be indented. Without indentation, Python does not know which code belongs to the condition.
  8. Q8: Cross out =, write == - SyntaxError: a single = is assignment (storing a value), while == is comparison (asking if two values are equal). Conditions require ==.
