Part 1: Variables and Basic Operations (Weeks 1-4)

1. Temperature Converter
Write a program that converts a temperature from Fahrenheit to Celsius. Your program should:
  • Ask the user to enter a temperature in Fahrenheit
  • Convert it to Celsius using the formula: C = (F - 32) * 5/9
  • Print the result with a descriptive message
Hint: Remember to convert the input to a float!
2. Age Calculator
Write a program that calculates how old someone will be in 10 years. Your program should:
  • Ask the user for their current age
  • Calculate their age in 10 years
  • Print a message showing their future age
Hint: Use int() to convert the input to an integer
3. Rectangle Area
Write a program that calculates the area of a rectangle. Your program should:
  • Ask the user for the length of the rectangle
  • Ask the user for the width of the rectangle
  • Calculate the area (length * width)
  • Print the result

Part 2: Strings (Week 3)

4. Name Formatter
Write a program that formats a person's name. Your program should:
  • Ask the user for their first name
  • Ask the user for their last name
  • Print their full name in the format: "Last, First" (all uppercase)
Hint: Use the .upper() method
5. Initials
Write a program that gets a person's initials. Your program should:
  • Ask the user for their first name
  • Ask the user for their last name
  • Print their initials (first letter of each name, uppercase)
Hint: Use indexing [0] and .upper()

Part 3: Conditionals (Week 5)

6. Grade Checker
Write a program that determines if a student passed or failed. Your program should:
  • Ask the user for a test score
  • If the score is 60 or above, print "Pass"
  • Otherwise, print "Fail"
7. Even or Odd
Write a program that determines if a number is even or odd. Your program should:
  • Ask the user for a number
  • If the number is even, print "Even"
  • If the number is odd, print "Odd"
Hint: Use the modulo operator (%) to check if a number is divisible by 2
8. Letter Grade
Write a program that converts a numeric score to a letter grade. Your program should:
  • Ask the user for a score (0-100)
  • Print "A" if score >= 90 , B, C etc.
  • Otherwise print "F"
Hint: Use elif statements

Part 4: Lists (Weeks 6-7)

9. List Statistics
Write a program that analyzes a list of numbers. Your program should:
  • Create a list: numbers = [10, 25, 30, 15, 40]
  • Print the length of the list
  • Print the maximum value in the list
  • Print the minimum value in the list
Hint: Use len(), max(), and min() functions
10. Shopping List
Write a program that manages a shopping list. Your program should:
  • Create an empty list called shopping_list
  • Add "milk" to the list
  • Add "bread" to the list
  • Add "eggs" to the list
  • Print the complete shopping list
  • Print how many items are on the list
Hint: Use .append() method

Part 5: For Loops (Week 8)

11. Countdown
Write a program that counts down from 5 to 1. Your program should:
  • Use a for loop to print numbers from 5 down to 1
  • Print "Blastoff!" after the countdown
Hint: Use range(5, 0, -1)
12. Sum Calculator
Write a program that calculates the sum of numbers from 1 to 10. Your program should:
  • Create a variable called total and set it to 0
  • Use a for loop to add each number from 1 to 10 to total
  • Print the final total
Hint: Use range(1, 11)
13. List Doubler
Write a program that doubles each number in a list. Your program should:
  • Create a list: numbers = [2, 4, 6, 8]
  • Use a for loop to print each number multiplied by 2

Part 6: While Loops (Week 9)

14. Password Checker
Write a program that keeps asking for a password until correct. Your program should:
  • Set a correct password (you choose the password)
  • Use a while loop to keep asking the user for the password
  • If the password is correct, print "Access granted!" and exit the loop
  • If incorrect, print "Try again" and ask again
Hint: The loop should continue while the password is incorrect
15. Number Guesser
Write a program that counts how many guesses it takes. Your program should:
  • Set a secret number (you choose the number)
  • Create a counter variable starting at 0
  • Use a while loop to keep asking the user to guess
  • Increment the counter each time they guess
  • When they guess correctly, print how many guesses it took
