Indexing

Access items by position. Starts at 0.

ExpressionGiven fruits = ["apple", "banana", "cherry"]Result
fruits[0]First item"apple"
fruits[2]Third item"cherry"
fruits[-1]Last item"cherry"
fruits[-2]Second to last"banana"

Slicing

list[start:end] - end index NOT included.

ExpressionGiven n = [10, 20, 30, 40, 50]Result
n[1:4]Index 1 to 3[20, 30, 40]
n[:3]From start to index 2[10, 20, 30]
n[2:]From index 2 to end[30, 40, 50]
n[:]Full copy[10, 20, 30, 40, 50]

Modifying Lists

Lists are mutable. Assign to an index to change an item.

fruits[1] = "mango"  # Changes "banana" to "mango"

Tuples

Immutable sequences created with parentheses. Cannot be changed after creation.

point = (10, 20)
x, y = point        # x = 10, y = 20
a, b = b, a         # Swap values
