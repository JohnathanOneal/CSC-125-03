Part 2: Explore the Python REPL

The REPL (Read-Eval-Print Loop) is Python's interactive mode. It evaluates code immediately, similar to Kaggle notebook cells but in your terminal.

Step 1: Open the REPL

Windows: Open Command Prompt or PowerShell, type python

Mac/Linux: Open Terminal, type python3

You should see the Python prompt: >>>

Try These Commands in the REPL
>>> print("Hello from my computer")
>>> 5 + 3
>>> name = "Alice"
>>> f"Hello, {name}"
>>> for i in range(5):
...     print(i)
...
>>> numbers = [1, 2, 3, 4, 5]
>>> sum(numbers)

REPL vs Notebooks

Similar: Both execute code interactively

Different: REPL is in your terminal, notebooks are in your browser

Different: REPL does not save your code automatically

Step 2: Exit the REPL

Type exit() or press Ctrl+D (Mac/Linux) or Ctrl+Z then Enter (Windows)

Checkpoint 2: REPL Works

You should have successfully opened the REPL, run some Python commands, and exited back to your normal terminal.

Part 3: Navigate the Command Line

Before running Python scripts, you need to understand basic terminal navigation. These commands work in Command Prompt (Windows), PowerShell (Windows), or Terminal (Mac/Linux).

Essential Commands

pwd (Mac/Linux) or cd with no arguments (Windows): Print Working Directory - shows where you are

ls (Mac/Linux) or dir (Windows): List files and folders in current location

cd foldername: Change Directory - move into a folder

cd ..: Move up one folder level

Practice: Where Am I?

Open your terminal and try these commands:

Windows (Command Prompt or PowerShell):

cd
dir

Mac/Linux (Terminal):

pwd
ls

The first command shows your current location. The second lists what is in that location.

Practice: Navigate to Desktop

Windows:

cd Desktop
dir

Mac/Linux:

cd Desktop
ls

You should now see a list of files and folders on your Desktop.

Understanding cd (Change Directory)

cd Desktop - moves into the Desktop folder

cd .. - moves up one level (back to parent folder)

cd Documents - moves into Documents folder

Think of folders like a tree. You are always "in" one folder, and you can move up or down.

Common Mistakes

Folder does not exist: If you get an error, the folder name might be wrong. Use dir or ls to see available folders.

Spaces in folder names: If a folder has spaces, use quotes: cd "My Documents"

Case sensitivity: On Mac/Linux, Desktop and desktop are different. Use exact capitalization.

Part 4: Run a Python Script from Terminal

Now that you can navigate folders, you can run Python scripts. Scripts are .py files you execute from the command line.

Step 1: Create a Python File

Open any text editor (Notepad on Windows, TextEdit on Mac, nano/vim on Linux)

Type this code:

print("This is my first script")
numbers = [10, 20, 30, 40, 50]
total = sum(numbers)
print(f"Total: {total}")

Save as test.py on your Desktop

Important for Mac users: In TextEdit, go to Format > Make Plain Text before saving

Step 2: Navigate to the File

In your terminal, navigate to Desktop where you saved the file:

Windows:

cd Desktop
dir

Mac/Linux:

cd Desktop
ls

You should see test.py in the list of files.

Step 3: Run the Script

Windows:

python test.py

Mac/Linux:

python3 test.py

You should see the output in your terminal.

What Just Happened?

  1. You navigated to the Desktop folder using cd
  2. You confirmed the file exists using dir or ls
  3. You ran the Python file using python or python3
  4. Python executed your code and displayed the output

Checkpoint 3: Script Execution Works

You should have:

  • Used cd to navigate to Desktop
  • Used dir or ls to verify the file exists
  • Created and run a .py file successfully

Bonus: Introduction to VS Code

VS Code is a popular code editor used by professional developers. It provides features like syntax highlighting, debugging, and extensions. This section is optional but recommended for future development.

Why Use VS Code?
  • Free and works on all operating systems
  • Syntax highlighting makes code easier to read
  • Built-in terminal
  • Git integration (for advanced GitHub workflows)
  • Extensions for Python, debugging, and more

Installing VS Code

Step 1: Download VS Code

Visit code.visualstudio.com

Download for your operating system

Step 2: Install

Windows: Run the installer, use default settings

Mac: Open the .dmg file, drag VS Code to Applications

Linux: Follow instructions for your distribution on the VS Code website

Step 3: Install Python Extension

Open VS Code

Click the Extensions icon on the left sidebar (four squares)

Search for "Python"

Install the extension by Microsoft

Using VS Code for Python

Create a Python File

File > New File

Save as example.py

Type some Python code:

print("Hello from VS Code")
numbers = [1, 2, 3, 4, 5]
for num in numbers:
    print(num * 2)
Run the Code

Right-click in the editor

Select "Run Python File in Terminal"

Or click the play button in the top right

VS Code Features to Explore

  • IntelliSense: Auto-completion as you type
  • Syntax Highlighting: Color-coded keywords and functions
  • Integrated Terminal: Run commands without leaving the editor
  • Debugging: Set breakpoints and step through code
