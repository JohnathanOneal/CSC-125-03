Advanced Slicing

list[start:end:step] - step controls how many items to skip.

ExpressionGiven n = [0, 10, 20, 30, 40, 50]Result
n[::2]Every other item[0, 20, 40]
n[::3]Every third item[0, 30]
n[1::2]Every other, starting at 1[10, 30, 50]
n[::-1]Reversed copy[50, 40, 30, 20, 10, 0]
n[4:1:-1]Backward from index 4 to 2[40, 30, 20]

Copying Lists

copy = original creates a reference, not a copy. Both names point to the same list.

safe = original.copy()   # Independent copy
safe = original[:]        # Also independent
safe = list(original)     # Also independent

String-List Conversion

MethodDirectionExampleResult
.split()String to list"a b c".split()["a", "b", "c"]
.split(",")String to list"a,b,c".split(",")["a", "b", "c"]

",".join(lst)List to string",".join(["a","b"])"a,b"

.join() requires all items to be strings. Convert numbers with str() first.

Nested Lists

Lists inside lists. Use double indexing: grid[row][col].

grid = [[1, 2], [3, 4], [5, 6]]
row = grid[0]       # [1, 2]
item = grid[0][1]   # 2
