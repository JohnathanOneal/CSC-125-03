Answer Key (Instructor Copy)

  1. Q1: b) 444 - input() returns a string, so "4" * 3 repeats the string three times
  2. Q2: c) student_id - hyphens (a), leading digits (b), and spaces (d) are not allowed in variable names
  3. Q3: b) 73 - both values are strings, so + concatenates them rather than adding numerically
  4. Q4: str - must convert the integer age to a string before concatenating with +
  5. Q5: int - converts the string "25" to a whole number so it can be added to 5
  6. Q6: Cross out age + 1, write int(age) + 1 - TypeError: cannot add a string and an integer
  7. Q7: Cross out 'red", write "red" or 'red' - SyntaxError: opening and closing quotes must match
  8. Q8: Cross out Favorite_Food, write favorite_food - NameError: Python variable names are case-sensitive
