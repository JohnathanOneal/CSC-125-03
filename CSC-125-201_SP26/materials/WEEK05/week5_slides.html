Why Programs Need Decisions

So far, every line of code runs every time. Real programs need to choose different actions based on conditions.

  • Is the user old enough to vote?
  • Did the student pass or fail?
  • Is the password correct?

Python uses if statements to run code only when a condition is True.

4
Section 2 · Lecture

The if Statement

The if keyword tests a condition. If True, the indented block runs.

age = 18

if age >= 18:
    print("You can vote")

print("Program continues")
  • The condition must end with a colon :
  • The body is indented by 4 spaces
  • Code after the block (unindented) always runs

if-else: Two-Way Decisions

Use else to handle the case when the condition is False. Exactly one block always runs.

number = 7

if number % 2 == 0:
    print("Even")
else:
    print("Odd")
  • else does not have a condition - it catches everything the if missed
  • else also needs a colon and an indented body

Section 2 · Lecture

if-elif-else: Multiple Options

Use elif (short for "else if") to check multiple conditions. The first True branch wins.

score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

Order matters - check from highest to lowest. Python stops at the first match.

Logical Operators

Combine conditions with and, or, and not:

ABA and BA or B
TrueTrueTrueTrue
TrueFalseFalseTrue
FalseTrueFalseTrue
FalseFalseFalseFalse
# and: BOTH must be True
# or:  at least ONE True
# not: flips the value

x = True
print(not x)  # False

Section 2 · Lecture

Combining Conditions

Use logical operators inside if statements to test multiple things at once:

# Both conditions must be True
age = 20
has_license = True

if age >= 16 and has_license:
    print("You can drive")
# At least one condition must be True
is_weekend = False
is_holiday = True

if is_weekend or is_holiday:
    print("Day off")

Section 2 · Lecture

Nested Conditionals

Place an if inside another if for decisions within decisions. Each level adds 4 more spaces.

age = 25
is_student = True

if age >= 18:
    if is_student:
        price = 8.00
    else:
        price = 12.00
else:
    price = 5.00

print(f"Ticket: ${price:.2f}")

Keep nesting to two levels when possible - deeper nesting gets hard to read.

Indentation Rules

Correct

if temperature > 90:
    print("It is hot")
    print("Stay hydrated")

if temperature < 32:
    print("Freezing")

Incorrect

# Missing colon
if temperature > 90
    print("It is hot")

# Inconsistent indent
if temperature < 32:
    print("Freezing")
      print("Wear a coat")
  • Always use 4 spaces per indentation level
  • Every if, elif, and else line must end with a colon
  • All lines in the same block must have the same indentation

Common Conditional Errors

ErrorCauseFix
SyntaxErrorMissing colon after conditionAdd : at end of if/elif/else
IndentationErrorWrong or inconsistent spacingUse exactly 4 spaces per level
Logic errorUsing = instead of === assigns, == compares
Logic errorWrong elif orderCheck from highest to lowest
# Wrong order - every score >= 70 also >= 60
if score >= 60:    # This catches 85 first
    grade = "D"    # Bug: 85 gets "D" not "B"
elif score >= 70:
    grade = "C"    # Never reached for 85

Summary

  • if runs code only when a condition is True
  • if-else provides a two-way choice - exactly one block runs
  • if-elif-else handles multiple options - first True wins
  • and, or, not combine conditions
  • Nested if statements handle decisions within decisions
  • Every condition line ends with a colon
  • Indentation is 4 spaces per level - consistency is required
