Basic if Statement

Runs the indented block only when the condition is True.

temperature = 95
if temperature > 90:
    print("It is hot outside")
print("Have a good day")  # Always runs

if-else Statement

Two branches. Exactly one block always runs.

number = 7
if number % 2 == 0:
    print(f"{number} is even")
else:
    print(f"{number} is odd")

if-elif-else Chain

Multiple conditions checked top to bottom. First True wins, rest are skipped.

score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"
print(f"Your grade is {grade}")

Combining Conditions

age = 25
has_license = True
if age >= 16 and has_license:
    print("You can drive")

day = "Saturday"
if day == "Saturday" or day == "Sunday":
    print("It is the weekend")

logged_in = False
if not logged_in:
    print("Please log in")

Nested Conditionals

Place an if inside another if. Each level adds 4 more spaces.

has_ticket = True
age = 15
if has_ticket:
    if age >= 18:
        print("Welcome to the R-rated movie")
    else:
        print("You need a parent for this movie")
else:
    print("Please purchase a ticket first")

Indentation Rules

  • Use 4 spaces per level (not tabs)
  • A colon is required after every if, elif, and else
  • All lines in the same block must have the same indentation
