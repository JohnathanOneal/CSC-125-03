What is a Variable?

A variable stores data in your program. Think of it as a labeled box:

  • The box = memory space
  • The label = variable name
  • What's inside = the value
4
Section 2 · Lecture

Why Variables Matter

  • Reusability - use the same value in multiple places
  • Flexibility - change it once, updates everywhere
  • Readability - descriptive names make code clear
hero_name = "Spider-Man"
city = "New York"

print("The amazing", hero_name, "protects", city)
print(hero_name, "swings through", city)

Floats

Numbers with a decimal point

True or False (capital letters)

Variable Naming Rules

Required

  • Start with a letter or underscore
  • Cannot start with a number
  • Only letters, numbers, underscores
  • No spaces, no dashes
  • Cannot use Python keywords
  • Case sensitive

Examples

ValidInvalid
user_nameuser name
age22age
is_validis-valid

Section 2 · Lecture

Naming Best Practices

  • Descriptive: student_grade not sg
  • snake_case: first_name not firstName
  • Lowercase for variables, UPPERCASE for constants
  • Booleans: prefix with is_ or has_
student_name = "Alice"
final_grade = 87.5
is_passing = True
MAX_STUDENTS = 30

Section 2 · Lecture

Type Conversion

Convert between types with int(), float(), str()

This works

text = "25"
number = int(text)  # 25
number + 5          # 30

This breaks

text = "25"
text + 1      # TypeError
int("Hello")  # ValueError

Section 2 · Lecture

The input() Function

input() asks the user for information. It always returns a string.

name = input("What's your name? ")
print("Hello,", name)

For numbers, convert the result:

age = int(input("Enter your age: "))
height = float(input("Enter height: "))
print("Next year:", age + 1)
