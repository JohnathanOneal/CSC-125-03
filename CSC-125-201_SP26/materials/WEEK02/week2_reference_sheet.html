Quick Reference

Variables
# Creating variables
hero_name = "Spider-Man"        # string
power_level = 8.5              # float
years_active = 10              # integer
from_outer_space = True        # boolean

# Check data type
print(type(hero_name))         # <class 'str'>
print(type(power_level))       # <class 'float'>
The Four Data Types
TypeStoresExample
String (str)Text in quotes"Hello", 'blue'
Integer (int)Whole numbers42, -5, 0
Float (float)Decimal numbers3.14, 19.99
Boolean (bool)True/FalseTrue, False

Note: "100" (with quotes) is a string. 100 is an integer. 100.0 is a float.

Variable Naming Rules
# Valid names
my_variable = 5
user_age = 25
hero_name = "Batman"
is_valid = True

# Invalid names (will cause errors)
# 2nd_variable = 5        # cannot start with number
# my-variable = 5         # no dashes
# my variable = 5         # no spaces
Type Conversion
# String to number
age_text = "25"
age_number = int(age_text)      # 25 (integer)
price_text = "19.99"
price_number = float(price_text) # 19.99 (float)

# Number to string
score = 100
score_text = str(score)         # "100" (string)

# Important: not all conversions work
# int("Hello") will cause a ValueError
User Input
# Getting text input
name = input("What's your name? ")

# Getting number input (must convert)
age = int(input("Enter your age: "))
height = float(input("Enter height: "))

# Using the input
print("Hello", name)
print("Next year you'll be", age + 1)
Basic Arithmetic
x = 10
y = 3

addition = x + y        # 13
subtraction = x - y     # 7
multiplication = x * y  # 30
division = x / y        # 3.333...
