Part 1: Multiple Choice (16 points)

1. d) boolean
True and False are the two boolean values in Python.
2. c) class
class is a reserved keyword in Python and cannot be used as a variable name. The others are all valid.
3. b) "Programming"
"Python Programming" has index 7 at the "P" in "Programming". [7:] slices from index 7 to the end.
4. b) 2
The modulo operator % returns the remainder. 20 divided by 6 is 3 with a remainder of 2.
5. c) string
input() always returns a string, even if the user types a number. You must convert it with int() or float().
6. b) 15
Negative indexing counts from the end. nums[-1] is 20, nums[-2] is 15.
7. b) None
.append() modifies the list in place and returns None. This is a common source of bugs when students write x = list.append(item).
8. a) 0, 2, 4, 6
range(0, 8, 2) starts at 0, increments by 2, and stops before 8. So: 0, 2, 4, 6.

Part 2: Fill in the Code (12 points)

9. int
int("30") converts the string "30" to the integer 30, allowing age + 1 to work.
10. upper
.upper() returns a new string with all characters converted to uppercase.
11. pop
.pop() removes and returns the last item from the list.
12. 1 : 3
Slicing is [start:end] where end is excluded. To get indices 1 and 2, use data[1:3].
13. 5 , 0 , -1
range(5, 0, -1) counts from 5 down to 1 (stops before 0). The negative step makes it count backward.
14. float
float(input(...)) converts the string input to a decimal number.

Part 3: Fix the Errors (12 points)

15. Logic error (wrong type): Change score = "90" to score = 90
With quotes, "90" * 2 repeats the string, producing "9090". Without quotes, 90 * 2 does math, producing 180. Alternative fix: doubled = int(score) * 2.
16. SyntaxError: Add a colon: for num in numbers:
Python requires a colon at the end of for, if, elif, else, and while lines.
17. SyntaxError: Use double quotes: name = "It's a test"
The apostrophe in "It's" closes the single-quoted string early. Using double quotes avoids the conflict. Alternative: name = 'It\'s a test'.
18. IndexError: Change letters[3] to letters[2]
A list with 3 items has valid indices 0, 1, 2. Index 3 is out of range.
19. IndentationError: Indent the print line: print("Too hot")
The code block under an if statement must be indented (4 spaces).
20. Logic error: Change numbers.sort() to sorted(numbers)
.sort() modifies the list in place and returns None. sorted() returns a new sorted list. The fix: sorted_list = sorted(numbers).

Part 5: Short Code Writing (15 points)

27. Filter a list:
scores = [72, 85, 68, 91, 77, 80] high_scores = [] for score in scores: if score >= 80: high_scores.append(score) print(high_scores)
Output: [85, 91, 80] - The loop checks each score and only appends those that are 80 or above.
28. Age category:
age = 16 if age < 13: print("Child") elif age <= 17: print("Teen") else: print("Adult")
Output: Teen (16 is not under 13, but 16 <= 17 is True)
29. Parallel lists with range(len()):
cities = ["Boston", "Miami"] pops = [700000, 450000] for i in range(len(cities)): print(f"{cities[i]}: {pops[i]}")
Output:
Boston: 700000
Miami: 450000

30. BMI Calculator (12 points):
weight = float(input("Enter weight in pounds: ")) height = float(input("Enter height in inches: ")) bmi = (weight / height ** 2) * 703 print(f"BMI: {bmi:.1f}") if bmi < 25: print("Healthy range") else: print("Above healthy range")
Scoring: input with conversion (3 pts), BMI formula (3 pts), formatted print (2 pts), if-else conditional (4 pts).

expenses = [12.50, 45.00, 8.75, 67.30, 23.99] total = 0 for expense in expenses: total = total + expense average = total / len(expenses) count = 0 for expense in expenses: if expense > 20: count = count + 1 print(f"Total: ${total:.2f}") print(f"Average: ${average:.2f}") print(f"Expenses over $20: {count}")
