Section 1 · Context

Why Learn the Fundamentals?

Personal example: I want to build a mobile app with a React Native frontend. I plan to use AI to help me build it. But before I do any of that, I am working through a tutorial like javascript.info/variables to learn the basics on my own.

The reason is simple. If I skip straight to AI, I cannot tell when it gives me something wrong. I cannot design the system. I cannot spot bad patterns. The AI becomes useless if I do not have the knowledge to evaluate what it produces.

Same idea here. We build up the fundamentals first so that when you do use AI tools, you know enough to work with them instead of just copying and hoping.

Section 2 · Lecture

String Methods

Built-in tools for cleaning and modifying text. Methods are called with dot notation:

text = "  HELLO world  "
text.strip()    # removes leading/trailing spaces
text.upper()    # all uppercase
text.lower()    # all lowercase
text.title()    # capitalize each word

Methods return a new string - the original is unchanged.

5
Section 2 · Lecture

Method Chaining & replace()

Chaining

Apply multiple methods in sequence:

name = "  alice JOHNSON  "
clean = name.strip().title()
# "Alice Johnson"

replace()

Swap text within a string:

msg = "I want EXTRA cheese"
msg.replace("EXTRA", "extra")
# "I want extra cheese"

Section 2 · Lecture

String Indexing

Each character has a numbered position, starting at 0:

word = "Python"
word[0]   # "P"  (first character)
word[5]   # "n"  (last character)
word[-1]  # "n"  (last, counting from end)
word[-2]  # "o"  (second to last)
7
Section 2 · Lecture

String Slicing

Extract multiple characters with [start:end]. The end index is not included.

sentence = "Hello World"
sentence[0:5]   # "Hello"
sentence[6:11]  # "World"
sentence[:5]    # "Hello"  (from start)
sentence[6:]    # "World"  (to end)
8
Section 2 · Lecture

Practical Slicing

Use .index() to find positions, then slice:

# Extract parts of an email
email = "student@college.edu"
at_pos = email.index("@")
username = email[:at_pos]       # "student"
domain = email[at_pos + 1:]     # "college.edu"
# Get initials from a name
name = "Alice Johnson"
space = name.index(" ")
initials = name[0] + "." + name[space + 1] + "."  # "A.J."

Section 2 · Lecture

F-String Formatting

Insert variables directly into strings with f"..." syntax:

Old way (concatenation)

name = "Alice"
age = 22
msg = "I am " + name + ", age " + str(age)

Modern way (f-strings)

name = "Alice"
age = 22
msg = f"I am {name}, age {age}"

F-strings can contain expressions: f"Next year: {age + 1}"

10
Section 2 · Lecture

F-Strings with Methods & Formatting

# Call methods inside f-strings
name = "alice"
print(f"Hello {name.title()}")   # "Hello Alice"
print(f"HELLO {name.upper()}")   # "HELLO ALICE"
# Format decimal places with :.2f
price = 19.99
tax = price * 0.08
print(f"Tax: ${tax:.2f}")        # "Tax: $1.60"
print(f"Total: ${price + tax:.2f}")  # "Total: $21.59"

Section 2 · Lecture

Reading Error Messages

Error messages tell you three things:

  • Error type - what went wrong (last line)
  • Location - which line (the arrow ---->)
  • Description - details about the problem

Section 2 · Lecture

Debugging Strategies

  • Step 1: Read the error message (last line first)
  • Step 2: Find the line number indicated by the arrow
  • Step 3: Add print() statements to check values
  • Step 4: Test with simple inputs before complex ones
# Debugging example
email = "student@college.edu"
print(f"Length: {len(email)}")      # check length
print(f"Type: {type(email)}")       # check type
at_pos = email.index("@")
print(f"@ at position: {at_pos}")   # check intermediate value
